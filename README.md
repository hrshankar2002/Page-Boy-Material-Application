# Page Boy Material Application
## Description
```
A flutter material app with realtime firebase backend services to keep track of all errands of a community.
```
## Getting Started

### Dependencies
```
* firebase_core: ^2.4.1
* firebase_auth: ^4.2.5
* cloud_firestore: ^4.3.1
* flutter_launcher_icons: ^0.11.0
```
